# Add FastAPI API Server (Keep CLI)

## Goal

Add a FastAPI REST API to the agent runtime while preserving the existing CLI. Both share the same runtime code (`run_agent`, repos, models). The API is the primary interface; the CLI is a convenience wrapper.

## Architecture

```
main.py                          CLI (unchanged)
src/agent_runtime/
  api/
    __init__.py
    app.py                       FastAPI app (lifespan, CORS, routers)
    schemas.py                   Pydantic request/response models
    deps.py                      Dependency injection (get_db, get_repo)
    routers/
      __init__.py
      sessions.py                POST/GET sessions, messages
      prompts.py                 GET/POST prompts, switch
  agents/runtime.py              Shared — used by both CLI and API
  db/                            Shared — repos, models, connection
  ...
```

Both CLI and API import from the same `agent_runtime.agents.runtime` and `agent_runtime.db.*` modules. No duplication.

## API Endpoints

### Sessions

| Method | Path                        | Description                          |
|--------|-----------------------------|--------------------------------------|
| POST   | /sessions                   | Create a new session                 |
| GET    | /sessions                   | List sessions (paginated)            |
| GET    | /sessions/{encoded_id}      | Get session details + messages       |
| POST   | /sessions/{encoded_id}/chat | Send message, get agent response     |

### Prompts

| Method | Path                        | Description                          |
|--------|-----------------------------|--------------------------------------|
| GET    | /prompts                    | List all system prompts              |
| POST   | /prompts                    | Create a new prompt                  |
| POST   | /sessions/{encoded_id}/prompt | Switch prompt for a session        |

### System

| Method | Path | Description        |
|--------|------|--------------------|
| GET    | /    | Health check       |

## Request/Response Shapes

### POST /sessions
```json
// Request (optional body)
{ "title": "My Chat" }

// Response 201
{ "id": "UgkBMxof", "title": "My Chat", "created_at": "2026-06-07T10:00:00" }
```

### POST /sessions/{id}/chat
```json
// Request
{ "message": "What's the weather in Jakarta?" }

// Response 200
{
  "response": "The weather in Jakarta is...",
  "session_id": "UgkBMxof",
  "messages": [
    { "id": 1, "role": "user", "content": "What's the weather...", "tool_name": null },
    { "id": 2, "role": "tool", "content": "[calling get_weather...]", "tool_name": "get_weather" },
    { "id": 3, "role": "tool", "content": "Weather in Jakarta...", "tool_name": "get_weather" },
    { "id": 4, "role": "assistant", "content": "The weather in Jakarta is...", "tool_name": null }
  ]
}
```

### GET /sessions
```json
// Query params: ?limit=20&offset=0

// Response 200
{
  "sessions": [
    { "id": "UgkBMxof", "title": "My Chat", "prompt": "default", "updated_at": "..." }
  ],
  "total": 5
}
```

### GET /sessions/{id}
```json
// Response 200
{
  "id": "UgkBMxof",
  "title": "My Chat",
  "prompt": "pirate",
  "created_at": "...",
  "messages": [
    { "id": 1, "role": "system", "content": "You are...", "tool_name": null },
    { "id": 2, "role": "user", "content": "Hello", "tool_name": null }
  ]
}
```

### GET /prompts
```json
// Response 200
{
  "prompts": [
    { "id": 1, "name": "default", "content": "You are a helpful assistant..." },
    { "id": 2, "name": "pirate", "content": "You are a pirate. Arr!" }
  ]
}
```

### POST /prompts
```json
// Request
{ "name": "pirate", "content": "You are a pirate. Arr!" }

// Response 201
{ "id": 2, "name": "pirate", "content": "You are a pirate. Arr!" }
```

### POST /sessions/{id}/prompt
```json
// Request
{ "name": "pirate" }

// Response 200
{ "prompt": "pirate" }
```

## Step-by-Step Plan

### Step 1: Add dependencies
```bash
uv add fastapi uvicorn[standard]
```

### Step 2: Create `src/agent_runtime/api/schemas.py`
Pydantic models for all request/response shapes listed above.
- `ChatRequest(message: str)`
- `ChatResponse(response: str, session_id: str, messages: list[MessageOut])`
- `SessionCreate(title: str = "New Session")`
- `SessionOut(id: str, title: str, prompt: str, created_at, updated_at)`
- `SessionDetail(id, title, prompt, created_at, messages)`
- `MessageOut(id, role, content, tool_name)`
- `PromptCreate(name: str, content: str)`
- `PromptOut(id, name, content)`
- `PromptSwitch(name: str)`

### Step 3: Create `src/agent_runtime/api/deps.py`
FastAPI dependency injection for the Database and repos:
```python
async def get_db() -> Database:
    """Reuse the same singleton from runtime.py."""
    from agent_runtime.agents.runtime import get_db as _get_db
    return await _get_db()

async def get_session_repo(db = Depends(get_db)) -> SessionRepo:
    return SessionRepo(db)

async def get_prompt_repo(db = Depends(get_db)) -> SystemPromptRepo:
    return SystemPromptRepo(db)
```

### Step 4: Create `src/agent_runtime/api/routers/sessions.py`
Routes:
- `POST /sessions` — create session (optionally with title and/or prompt_name)
- `GET /sessions` — list sessions with prompt names
- `GET /sessions/{encoded_id}` — get session detail with messages
- `POST /sessions/{encoded_id}/chat` — send message, call `run_agent()`, return response + new messages

For `/chat`, call `run_agent(message, session_id)` from `agent_runtime.agents.runtime`. Then query messages added after the user message to return only the new ones (or return all — simpler).

### Step 5: Create `src/agent_runtime/api/routers/prompts.py`
Routes:
- `GET /prompts` — list all prompts
- `POST /prompts` — create new prompt
- `POST /sessions/{encoded_id}/prompt` — call `switch_prompt()`

### Step 6: Create `src/agent_runtime/api/app.py`
```python
from contextlib import asynccontextmanager
from fastapi import FastAPI
from fastapi.middleware.cors import CORSMiddleware

@asynccontextmanager
async def lifespan(app: FastAPI):
    # Startup: connect DB, seed default prompt
    db = await get_db()
    prompt_repo = SystemPromptRepo(db)
    await prompt_repo.seed_default()
    yield
    # Shutdown: disconnect DB
    await db.disconnect()

app = FastAPI(title="Agent Runtime", version="0.1.0", lifespan=lifespan)
app.add_middleware(CORSMiddleware, allow_origins=["*"], ...)
app.include_router(sessions_router)
app.include_router(prompts_router)
```

### Step 7: Add `api` command to `main.py`
```python
elif args[0] == "api":
    import uvicorn
    uvicorn.run("agent_runtime.api.app:app", host="0.0.0.0", port=8000, reload=True)
```

So: `uv run python main.py api` starts the server.

### Step 8: Add tests
- `tests/api/test_schemas.py` — validate Pydantic models
- `tests/api/test_sessions.py` — test endpoints with httpx.AsyncClient + TestClient
- `tests/api/test_prompts.py` — test prompt endpoints

Use `httpx.AsyncClient` with `ASGITransport` for async testing:
```python
from httpx import ASGITransport, AsyncClient
from agent_runtime.api.app import app

async with AsyncClient(transport=ASGITransport(app=app), base_url="http://test") as client:
    resp = await client.get("/prompts")
```
